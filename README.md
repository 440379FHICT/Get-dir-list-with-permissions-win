# Get-dir-list-with-permissions-win
Deze oplossig is om de directories en de permissies van alle mappen die onder het aangegeven pad zitten in windows.
Deze oplossig bestaat uit twee scripts en een map Folder.ps1, Scheduler.py en Output.

Folder.ps1
Zorg ervoor dat jouw Windows systeem de rechten heeft om powershell script uit te voeren.
Om het pad aan te passen, bewerk het bestand en pas de waarde onder $path aan.
